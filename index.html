Frequently asked questions

What is FastGraph Swarm Builder?

It’s a no‑code way to build swarms of LLM agents from plain‑language prompts, orchestrate them into workflows, and connect them to the tools you already use.

Can non‑technical users create and evolve agents?

Yes. Business users can create agents with prompts and iterate using guided evaluation loops—no code required.

How do guardrails and approvals work?

You can define policies (allow/deny, PII controls, tool permissions) and require approvals for sensitive actions. All changes are auditable.

What integrations are supported?

We focus on practical connectors: CRMs, ticketing, monitoring, data sources, and model providers. Tell us your top tools—we’ll prioritize them.

How do we go from demo to production?

Start with a guided demo, then enable projects, approvals, and monitoring. Self‑host and enterprise options are planned; contact us for timelines.

What is EvolSpace?

EvolSpace is our managed Agent Evolution service—run evaluations, compare outcomes, and apply guardrail‑aware updates without managing infra.
